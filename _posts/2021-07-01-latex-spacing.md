| Latex code | Description                                    |
|------------|------------------------------------------------|
| `\quad`    | space equal to the current font size (= 18 mu) |
| `\,`       | 3/18 of \quad (= 3 mu)                         |
| `\:`       | 4/18 of \quad (= 4 mu)                         |
| `\;`       | 5/18 of \quad (= 5 mu)                         |
| `\!`       | -3/18 of \quad (= -3 mu)                       |
| `\ `       | equivalent of space in normal text             |
| `\qquad`   | twice of \quad (= 36 mu)                       |
